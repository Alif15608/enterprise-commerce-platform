# Enterprise Commerce Platform

A production-grade, full-stack e-commerce platform built to demonstrate
clean architecture, scalable backend design, and modern DevOps practices.

## Status
🚧 In active development — built in public, phase by phase.
See [docs/adr](docs/adr) for architecture decisions.

## Tech Stack
- **Backend**: Django, Django REST Framework, FastAPI, PostgreSQL, Redis, Celery
- **Frontend**: React, TypeScript, Redux Toolkit, Tailwind CSS
- **Infra**: Docker, GitHub Actions, Terraform, Nginx
- **Observability**: Prometheus, Grafana
